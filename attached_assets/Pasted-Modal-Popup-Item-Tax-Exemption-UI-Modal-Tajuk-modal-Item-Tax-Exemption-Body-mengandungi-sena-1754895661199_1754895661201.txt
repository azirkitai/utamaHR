Modal Popup – “Item/Tax Exemption”
UI Modal:

Tajuk modal: Item/Tax Exemption

Body mengandungi senarai input amount RM untuk kategori pengecualian cukai LHDN, contoh:

Travelling Allowance

Child Care Allowance

Gift

Phone Allowance

Reward

Parking Allowance

Meal Allowance

Subsidies

Setiap item:

Label nama

Input RM (2 d.p.)

Ikon gear kecil (⚙) untuk buka setting flags statutory (optional jika mahu konsisten dengan sistem).

Bawah senarai, butang Close (warna merah) untuk tutup modal.

3. Fungsi Modal
Bila modal dibuka, nilai input prefill ikut data yang tersimpan dalam taxExemptions[] untuk pekerja tersebut.

Bila user masukkan nilai, sistem simpan ke taxExemptions[] (on change atau on save).

Nilai dari modal tidak menambah kepada gross salary kerana ini adalah pengecualian cukai, tetapi digunakan untuk pengiraan PCB (mengurangkan taxable income).

Pastikan semua nilai ini tidak termasuk dalam wage base EPF, SOCSO, EIS, HRDF (melainkan HR override manual).

4. Integrasi Dengan PCB39
PCB39 calculation perlu tolak jumlah dari semua item dalam taxExemptions[].

PCB taxable income = (Basic Salary + Additional Items taxable) – (Total Tax Exemptions).
